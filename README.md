# Smart Inventory Management System

## Project Description

A web-based inventory and sales management system designed to help small retail stores track stock, record sales, and analyze product performance.

## Features

- Add and manage products
- Record sales transactions
- Automatic stock updates
- Dashboard analytics
- Revenue tracking
- Sales visualization using charts

## Technology Stack

Frontend
- HTML
- CSS
- Chart.js

Backend
- Python
- Flask

Database
- SQLite

ORM
- SQLAlchemy

## How to Run the Project

1. Clone the repository

git clone https://github.com/yuvi2307/Techrossum.git

2. Go to the project folder

cd smart-inventory-management-system

3. Install dependencies

pip install -r requirements.txt

4. Run the application

python app.py

5. Open in browser

http://127.0.0.1:5000

## Note

This project runs locally on localhost for demonstration purposes.
